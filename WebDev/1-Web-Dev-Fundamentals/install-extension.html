How to Install and Use VS Code's Live Server Extension

Install | Use | Video

Installation Instructions

  1. In Visual Studio Code, click the Extensions icon in the left column toolbar.
  2. In the search bat that appears, type Live Server
  3. Select the Liver Server extension. It should be the top choice.
  4. In the page for Live Server the appears, click the green Install button.

Usage Instructions

  • Visual Studio Code must be open to a folder to use Live Server.
  • In VS Code's Explorer, right click (Windows) or Control+click (macOS) on an HTML file.
